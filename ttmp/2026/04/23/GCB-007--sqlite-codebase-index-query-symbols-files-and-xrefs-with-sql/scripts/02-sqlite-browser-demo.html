🗄️ GCB-007: SQLite + TinyGo WASM Demo

A standalone prototype proving that SQLite (via sql.js) and Go (via TinyGo WASM) can work together in the browser to query a codebase index. No server needed — open this file directly from file:// or any static server.

How this works:

  • sql.js — SQLite compiled to WASM, loaded from CDN. Provides the database engine.
  • TinyGo WASM — Go code compiled to WASM. Provides schema SQL, sample data, result formatting, and query suggestions.
  • FTS5 — SQLite's full-text search extension. Tokenizes symbol names, docs, and signatures. Ranked with BM25.
  • Open browser DevTools console to see window.goQuery exports and window.db.
