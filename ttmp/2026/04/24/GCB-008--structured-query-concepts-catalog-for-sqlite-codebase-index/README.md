# Structured Query Concepts Catalog for SQLite Codebase Index

This is the document workspace for ticket GCB-008.

## Goal

Build a SQL-only structured concept catalog for `codebase-browser` so reusable SQLite queries become named, typed CLI verbs and can later become generated web UI forms.

## Structure

- **design-doc/**: implementation guide and architecture notes
- **reference/**: diary and reference material
- **playbooks/**: validation and command runbooks
- **scripts/**: temporary scripts
- **various/**: scratch notes
- **archive/**: deprecated/reference-only material
